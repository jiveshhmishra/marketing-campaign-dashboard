# Marketing Campaign Analysis

## Overview

This project focuses on analyzing marketing campaign data using Python and visualizing insights through a Power BI dashboard.

## Key Insights

* Social Media and Display Ads have higher conversion rates
* South and East regions generate more purchases
* Younger age group (18–30) is more active
* Discounts increase customer purchases

## Tools Used

* Python (Pandas)
* Jupyter Notebook
* Power BI

## Files

* marketing_campaign_analysis.ipynb → Data analysis and cleaning
* market.csv → Dataset
* Marketing Campaign Analysis Dashboard.pbix → Power BI dashboard

## Conclusion

## Conclusion

The analysis shows that Social Media and Display Ads perform better in terms of conversions. South and East regions generate more purchases, and younger customers are more active. It is also observed that higher discounts increase purchasing behavior.

Overall, this project helps in understanding customer trends and improving marketing strategies.
